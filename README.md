# 音楽と言葉の記録サイト

**「ある言葉に、なぜ惹かれたか」を残す記録** 

このプロジェクトは、歌詞の意味を解説するのではなく、**なぜその言葉に惹かれたのか**を記録するためのサイトです。

## プロジェクトの位置づけ

- 歌詞の意味を説明する場所ではない
- 正解を提示する場所でもない
- **「ある言葉に、なぜ惹かれたか」を残す記録**

他人に向けて"売る"のではなく、自分のために"残す"。結果として、同じ温度の人が読む。

## 技術スタック

- **Next.js** - フレームワーク
- **Vercel** - ホスティング
- **Markdown** - 記事管理

## 目標

- **アーカイブとして完成している状態を目指す**
- 1年後、記事が30本あれば、それ自体が価値
- 検索に引っかかるようになるのは、記事が溜まってから

# SEO強化マニュアル - ページ別検証・改善ガイド

**目標**: 各ページで、必ず1つ以上のキーワード（または組み合わせ）で検索1位を取る

---

## 📋 基本方針

### 1. 検索1位を取るための3原則
1. **ニッチ×具体性**: 競合が少なく、検索意図が明確なキーワードを狙う
2. **独自性**: 他のサイトにない情報・視点を必ず含める
3. **検索意図の完全充足**: 検索者が「これで解決した」と思える情報量

### 2. 対象ページ
- トップページ (`/`)
- 記事ページ (`/posts/[slug]`)
- アーティストページ (`/artists/[name]`)
- 固定ページ (`/about`, `/contact`, `/privacy`)


## 📋 既存記事の構造チェック＆修正ガイド

### なぜ構造が重要か？

記事の構造（H2/H3の階層と`<span class="section-subtitle">`タグ）は、以下の理由で重要：

1. **目次生成**: TableOfContentsコンポーネントがH2/H3から目次を生成
2. **ユーザー体験**: 目次がないと読者が記事の全体像を把握できない
3. **SEO**: 構造化された見出しは検索エンジンの理解を助ける

### ステップ1: 記事構造の診断（5分）

**必須チェック項目:**

```bash
# H2の数を確認
grep -c "^## " content/posts/[記事名].md

# H3の数を確認
grep -c "^### " content/posts/[記事名].md

# H2にタグがない行を検出（これが出力されたらNG）
grep "^## [^<]*$" content/posts/[記事名].md

# H3にタグがない行を検出（これが出力されたらNG）
grep "^### [^<]*$" content/posts/[記事名].md
```

**診断基準:**

- ✅ **合格**: すべてのH2に最低1つのH3がある、すべてにタグがある
- ⚠️ **要修正**: H3が0個のH2セクションがある、タグが抜けている
- ❌ **不合格**: H3が全くない、タグが全て抜けている

---

### ステップ3: 修正の優先順位

**優先度A（即修正）:**
- アクセス数が月100PV以上
- Google検索で11〜30位（あと少しで1ページ目）
- 独自性が高く、競合が少ない記事

**優先度B（次回修正）:**
- アクセス数が月50PV以上
- 関連記事からの内部リンクが多い記事

**優先度C（余裕があれば）:**
- その他の記事
- 映像作品の記事（楽曲記事を優先）

## 📝 ページ別改善フロー

### 新規記事 or コンテンツ強化の改善手順

#### 1. 現状確認（所要時間: 15分）
```
□ タイトルを読む → キーワードを特定
□ ディスクリプションを確認
□ 見出し構造を確認（H2, H3）
□ 文字数をカウント
□ 内部リンク数を確認
```

#### 2. キーワード調査（所要時間: 20分）
```
□ Google検索でメインキーワードを検索
□ 上位10サイトの内容をメモ
□ 関連検索キーワードを確認
□ サジェストキーワードを収集
```

**使用ツール**:
- Google検索
- ラッコキーワード（https://related-keywords.com/）
- Ubersuggest（https://neilpatel.com/ubersuggest/）

#### 3. 改善案作成（所要時間: 10分）
```
□ タイトル修正案を3つ作成
□ ディスクリプション修正案を作成
□ 追加すべき小見出しをリストアップ
□ 関連記事リンク（relatedPosts）を3〜5記事設定
□ 追加すべき情報を特定
```

#### 4. 実装（所要時間: 30分〜1時間）
```
□ タイトル・ディスクリプション修正
□ keywords配列の追加（カタカナ版を必ず含める）
□ 見出しの追加・修正
□ 本文の加筆・修正
□ 内部リンクの追加（relatedPosts: 3〜5記事）
□ 画像のalt属性設定
```

**relatedPostsの設定基準:**
- 同じアーティストの他の曲記事
- テーマが関連する記事（例：結婚式ソング同士、失恋ソング同士）
- ドキュメンタリーや映画記事（該当する場合）
- 3〜5記事を設定（多すぎても少なすぎてもNG）

#### 5. 検証（所要時間: 10分）
```
□ プレビューで表示確認
□ モバイル表示確認
□ 内部リンクの動作確認
□ Search Console にインデックス申請
```

## 🚀 優先順位付け

```

```

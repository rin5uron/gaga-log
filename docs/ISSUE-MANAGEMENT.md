# Issue管理 - gaga-log

## 基本原則

**アイデア・作業履歴・進捗は全てIssueで管理**

- ドキュメントは `docs/` に集約（これ以上増やさない）
- それ以外の情報（アイデア、改善案、作業履歴）は GitHub Issue で管理
- Issue コメントで時系列の記録を残す

---

## タイトル形式

**形式**: `【カテゴリ】対象 - 作業内容`

| カテゴリ | 使う場面 |
|----------|---------|
| `【リライト】` | 記事の書き直し・テンプレ準拠 |
| `【新記事】` | 新規記事作成 |
| `【SEO】` | SEO改善・CTR改善 |
| `【AdSense】` | AdSense審査・収益化対応 |
| `【設計】` | ルール・テンプレ・設計書 |
| `【デザイン】` | UI/UXデザイン |
| `【バグ】` | 不具合修正 |
| `【インフラ】` | サイト構造・技術実装 |
| `【学習】` | Next.js/Tailwind学習 |
| `【記録】` | 作業メモ・実装記録 |

**例**:
- `【リライト】Judas - テンプレ準拠・考察追加`
- `【リライト】Alejandro - H2追加・文体修正`
- `【SEO】Alejandro - CTR改善`
- `【AdSense】全記事 - H2構造変更`
- `【設計】リライトルール`
- `【バグ】Vercelビルド失敗 - layout.tsx img属性`

---

## ラベル体系

| ラベル | 用途 |
|--------|------|
| `content` | 記事執筆・リライト関連 |
| `seo` | 検索エンジン最適化 |
| `adsense` | AdSense収益化対応 |
| `design` | UI/UXデザイン |
| `infrastructure` | サイト構造・技術実装 |
| `learning` | Next.js/Tailwind学習 |
| `bug` | 不具合 |
| `documentation` | ドキュメント・設計書 |
| `security` | セキュリティ関連 |
| `analytics` | Google Analytics関連 |

- **必ず1つ以上付ける**（複数可）
- 記事関連: `content` + `seo` など
- サイト全体: `infrastructure` + `design` など

## Issueクローズのタイミング

- チェックリストがすべて完了したらクローズ
- 新たな問題が発生したら新しいIssueを作成（既存issueに追記しない）

---

## 作業の流れ

### リライト作業（#22・#30・個別issueの3点セット）
1. **#22**（全体方針）・**#30**（リライトルール）を読み込む
2. 対象記事のissueを開き、チェックリストに沿って作業
3. H2変更・セクション順序修正・frontmatter（theme）追加を同時に行う
4. 完了後、issueコメントに記録してクローズ

### 新規記事作成
1. `/new-article` 実行
2. Issue作成（`【新記事】曲名`、ラベル: `content`）
3. 公開後クローズ

### SEO改善
1. `/seo-check` 実行
2. Issue作成（`【SEO】曲名 - 作業内容`、ラベル: `seo,content`）
3. 完了後クローズ

### バグ修正
1. Issue作成（`【バグ】内容`、ラベル: `bug`）
2. 修正完了後クローズ

### 記録・確認作業
1. Issue作成（`【記録】対象 - 作業内容`、ラベル: `content` など該当）
2. 本文に背景・内容・結果を記載
3. 完了後コメントに日付を書いてクローズ

## 記録用Issueの例（コピペ用）

**※ 以下の内容でGitHubで新規Issueを作成し、記録として残す。CLIで作成する場合は `gh issue create --body-file ファイル名` を使う。**

**テンプレのルール**: **日付は必ず本文の一番上に「記録日」として書く。**

### タイトル
```
【記録】全記事 - YouTube埋め込み公式動画確認
```

**ラベル**: `content` または `bug`（修正が入ったため）

### 本文（記録日を一番上に）
```markdown
## 記録日
- YYYY-MM-DD（実施した日）

---

## 内容
- [x] 全35記事のYouTube埋め込み動画IDを洗い出し
- [x] 各曲の公式MV/公式チャンネルの動画IDを検索で照合
- [x] 非公式だった1件を公式動画に差し替え

---
```

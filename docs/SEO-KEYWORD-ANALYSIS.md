## 最適化提案

### 1. keywordsフィールドの改善

**現在:**
```
keywords: "レディーガガ, ガガ, ライブレポート, 東京ドーム, MAYHEM Ball, 2026, 来日公演, Lady Gaga, 会場情報, ライブ当日, ネタバレなし、メイヘム"
```

**提案:**
```
keywords: "レディーガガ, ガガ, ライブレポート, 東京ドーム, 雰囲気, 空気感, 服装, 持ち物, 会場, 当日, 参戦, 初日, 2026, MAYHEM Ball, 来日公演, Lady Gaga, 会場情報, ライブ当日, ネタバレなし, メイヘム, SS席, エキサイトシート, 感想, レポート"
```

### 2. titleの改善

**現在:**
```
title: "The MAYHEM Ball東京ドーム公演2026/1/25"
```

**提案:**
```
title: "ガガ ライブ 東京ドーム 雰囲気・服装・持ち物レポート 2026/1/25"
```

または、より自然に：
```
title: "The MAYHEM Ball東京ドーム公演レポート 2026/1/25 - 雰囲気・服装・当日の様子"
```

### 3. descriptionの改善

**現在:**
```
description: "The MAYHEM Ball東京ドーム公演（1/25）のライブレポート。ネタバレなし。会場の空気感、ファッション、荷物預けの話、これから行く人へのメモ。"
```

**提案:**
```
description: "ガガ ライブ 東京ドームの雰囲気・服装・持ち物を徹底レポート。会場の空気感、SS席エキサイトシートの様子、当日の持ち物・荷物預け情報まで。ネタバレなしでこれから参戦する人向け。"
```

### 4. 本文の見出し最適化

**現在の見出し:**
- `### 1-2. 会場の雰囲気・客層・ファッション`

**提案:**
- `### 1-2. ガガ ライブ 東京ドーム 雰囲気・客層・ファッション`

または、より自然に：
- `### 1-2. 会場の雰囲気・客層・ファッション（東京ドームの空気感）`

### 5. 本文内のキーワード自然な組み込み

以下のキーワードを自然に本文に組み込む：

- **「ガガ ライブ 東京ドーム 雰囲気」** → 冒頭や見出し付近に
- **「ガガ ライブ 東京ドーム 服装」** → ファッションセクションに
- **「ガガ ライブ 東京ドーム 持ち物」** → 持ち物セクションに
- **「ガガ ライブ 東京ドーム 当日」** → 当日の様子セクションに
- **「ガガ ライブ 東京ドーム 空気感」** → 雰囲気セクションに

### 6. alt属性の最適化

画像のalt属性にキーワードを含める：
- `alt="ガガ ライブ 東京ドーム 会場の様子"`
- `alt="ガガ ライブ 東京ドーム SS席エキサイトシート"`

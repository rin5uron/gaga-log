# ■ ルールまとめ

**テーマ・構成**
・テーマは1記事1つ
・frontmatterに theme を置く
・ラベルは `<div class="section-label">〇〇 <span class="label-subtitle">〇〇</span></div>` の表記。## は各ラベル内のH2見出しにだけつける。各ラベルに ## H2 を2つずつ

**About**
・サブタイトル → 概要 → テーマのこと（どんな曲？で締める）→ 2つ目のH2で深掘り

**歌詞のポイント**
・テーマを証明する引用。その表現がテーマに沿っていれば語学メモもOK。語学メモは `<div class="language-note">` で囲み、あとからデザインしやすくする

**考察**
・1つ目のH2でテーマに沿って深掘り、2つ目のH2でまとめ

**口ずさみたいフレーズ**
・楽しさゾーン。**別セクションに置く**（考察のあと。このサイト読むと英語の勉強にもなる＝歌詞のポイントの語学メモで担保）

**Music Video**
・MVがある曲は、考察のあと（口ずさみの前）に置く。ラベルは `<div class="section-label">Music Video <span class="label-subtitle">MVの見どころ</span></div>`

**文体**
・だ調は About の概要（リリース情報の段落）だけ。それ以外はです・ます調で統一する

**References**
・毎回同じ構成（公式MV・インタビュー・その他など）。テンプレに合わせる

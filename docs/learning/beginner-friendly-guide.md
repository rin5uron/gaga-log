## 準備するもの

### 必要なもの

1. **パソコン**（Mac、WindowsどちらでもOK）
2. **エディタ**（コードを書くソフト）
   - VS Code（おすすめ）
   - または、Cursor（今使っているもの）
3. **ブラウザ**（Chrome、Safari、Firefoxなど）
4. **開発サーバーが動いていること**

開発サーバーが動いているか確認する方法：

1. ターミナル（黒い画面）を開く

   ```
   - Local:        http://localhost:3000
   - Ready in 2.3s
   ```
3. ブラウザで `http://localhost:3000` を開いて、サイトが表示されればOKです！

## デザインを変えるってどういうこと？

### 例え話：お部屋の模様替え

デザインを変えるのは、**お部屋の模様替え**と同じです。

- **壁の色を変える** → 背景色を変える（`bg-white` など）
- **家具の位置を変える** → 余白を調整する（`p-6` など）
- **ラグを敷く** → 影を付ける（`shadow-md` など）
- **角の丸いテーブルを置く** → 角を丸くする（`rounded-xl` など）

### コードの見方

```tsx
<div className="bg-white rounded-xl shadow-md p-6">
```

このコードは、日本語にすると：

- `bg-white` = 背景を白にする
- `rounded-xl` = 角を丸くする（大きめ）
- `shadow-md` = 影を付ける（中くらい）
- `p-6` = 内側に余白を付ける（24px）

**スペースで区切って、いくつも並べることができます！**

## 角を丸くする方法

### 角を丸くするって？

四角い箱の角を、**丸く**することです。

### どうやって書く？

`rounded-` という言葉の後に、**サイズ**を書きます。

```tsx
rounded-sm    ← 少しだけ丸く（2px）
rounded       ← 普通に丸く（4px）
rounded-md    ← 中くらいに丸く（6px）
rounded-lg    ← 大きめに丸く（8px）
rounded-xl    ← もっと大きく丸く（12px）
rounded-2xl   ← すごく大きく丸く（16px）
rounded-full  ← 完全に丸く（円形になる）
```

### 実際にやってみよう

1. `ArticleHighlights.tsx` を開く
2. 13行目を見つける：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-lg">
   ```
3. `rounded-lg` を `rounded-xl` に変えてみる：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-xl">
   ```
4. ファイルを保存する（⌘+S または Ctrl+S）
5. ブラウザを見る → 自動で更新されます！

**角がもっと丸くなったのが見えますか？** ✨

## 影を付ける方法

### 影を付けるって？

箱が**浮いているように見える**効果です。

### どうやって書く？

`shadow-` という言葉の後に、**サイズ**を書きます。

```tsx
shadow-sm    ← 小さい影（1px）
shadow       ← 普通の影（標準）
shadow-md    ← 中くらいの影（4px）← カードにぴったり！
shadow-lg    ← 大きい影（10px）
shadow-xl    ← すごく大きい影（20px）
```

### 実際にやってみよう

1. `ArticleHighlights.tsx` を開く
2. 13行目を見つける：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-lg">
   ```
3. `shadow-md` を追加してみる：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-lg shadow-md">
   ```
4. ファイルを保存する
5. ブラウザを見る → 影が付いたのが見えます！

**箱が浮いているように見えませんか？** ✨

## 色を変える方法

### 背景の色を変える

`bg-` という言葉の後に、**色の名前**を書きます。

```tsx
bg-white      ← 白
bg-gray-50    ← 薄いグレー
bg-gray-100   ← 少し濃いグレー
bg-blue-50    ← 薄い青
bg-red-50     ← 薄い赤
```

### 文字の色を変える

`text-` という言葉の後に、**色の名前**を書きます。

```tsx
text-gray-700  ← 濃いグレー（読みやすい）
text-gray-600  ← 中くらいのグレー
text-blue-500  ← 青
text-red-500   ← 赤
```

### 実際にやってみよう

1. `ArticleHighlights.tsx` を開く
2. 13行目を見つける：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-lg">
   ```
3. `bg-gray-50` を `bg-white` に変えてみる：
   ```tsx
   <div className="mb-4 p-5 bg-white rounded-lg">
   ```
4. ファイルを保存する
5. ブラウザを見る → 背景が白くなりました！

**背景が白くなったのが見えますか？** ✨

## 余白を調整する方法

### 余白って？

箱の**内側**と**外側**の空いているスペースです。

### 内側の余白（パディング）

`p-` という言葉の後に、**数字**を書きます。

```tsx
p-0    ← 余白なし（0px）
p-1    ← 小さい余白（4px）
p-2    ← 少し大きい余白（8px）
p-4    ← 中くらいの余白（16px）
p-5    ← 大きい余白（20px）
p-6    ← もっと大きい余白（24px）← カードにぴったり！
```

### 外側の余白（マージン）

`m-` という言葉の後に、**数字**を書きます。

```tsx
m-0    ← 余白なし
m-4    ← 中くらいの余白（16px）
m-6    ← 大きい余白（24px）
mb-4   ← 下だけに余白（16px）
mt-4   ← 上だけに余白（16px）
```

### 実際にやってみよう

1. `ArticleHighlights.tsx` を開く
2. 13行目を見つける：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-lg">
   ```
3. `p-5` を `p-6` に変えてみる：
   ```tsx
   <div className="mb-4 p-6 bg-gray-50 rounded-lg">
   ```
4. ファイルを保存する
5. ブラウザを見る → 内側の余白が大きくなりました！

**箱の中の余白が大きくなったのが見えますか？** ✨

## 実際にやってみよう

### チャレンジ1: カード風デザインを作る

「この記事でわかること」を、**カード風**にしてみましょう！

#### 手順

1. `ArticleHighlights.tsx` を開く
2. 13行目を見つける：
   ```tsx
   <div className="mb-4 p-5 bg-gray-50 rounded-lg">
   ```
3. これを以下のように変える：
   ```tsx
   <div className="mb-6 p-6 bg-white rounded-xl shadow-md border border-gray-200">
   ```

#### 何が変わった？

- `mb-4` → `mb-6` = 下の余白を大きく
- `p-5` → `p-6` = 内側の余白を大きく
- `bg-gray-50` → `bg-white` = 背景を白に
- `rounded-lg` → `rounded-xl` = 角をもっと丸く
- `shadow-md` = 影を追加（新しく追加！）
- `border border-gray-200` = ボーダーを追加（新しく追加！）

4. ファイルを保存する
5. ブラウザを見る → カード風になりました！🎉

### チャレンジ2: ホバーエフェクトを追加する

マウスを乗せた時に、**影が大きくなる**ようにしてみましょう！

#### 手順

1. 先ほどのコードに、さらに追加する：
   ```tsx
   <div className="mb-6 p-6 bg-white rounded-xl shadow-md border border-gray-200 hover:shadow-lg transition-shadow duration-200">
   ```

#### 何が追加された？

- `hover:shadow-lg` = マウスを乗せた時に影を大きく
- `transition-shadow` = 影の変化をスムーズに
- `duration-200` = 200ミリ秒で変化

2. ファイルを保存する
3. ブラウザで、マウスを「この記事でわかること」の上に乗せる
4. 影が大きくなりました！✨

## 困ったときは

### Q: 変更が反映されない

**A: 確認すること：**

1. **ファイルを保存した？**
   - ⌘+S（Mac）または Ctrl+S（Windows）で保存
2. **開発サーバーが動いている？**
   - ターミナルに `▲ Next.js` と表示されているか確認
3. **ブラウザをリロードした？**
   - F5キーを押すか、⌘+R（Mac）または Ctrl+R（Windows）

### Q: エラーが出た

**A: 確認すること：**

1. **スペルミスはない？**
   - `rounded-xl` が `rounde-xl` になっていないか確認
2. **スペースは正しい？**
   - クラスとクラスの間は**スペース1つ**で区切る
3. **引用符は閉じている？**
   - `className="..."` の引用符が閉じているか確認

### Q: どのクラスを使えばいいかわからない

**A: まずはこれから始めよう：**

- 角丸: `rounded-xl`（大きめに丸く）
- 影: `shadow-md`（中くらいの影）
- 背景: `bg-white`（白）
- 余白: `p-6`（内側）、`mb-6`（下）

これらを組み合わせれば、カード風デザインが作れます！

### Q: もっと詳しく知りたい

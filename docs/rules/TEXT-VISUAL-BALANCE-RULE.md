# 文字と視覚要素のバランスルール

**作成日**: 2026-01-30
**目的**: 文字ばかりで見づらくならないよう、適度に視覚要素を挿入する

---

## 🎯 基本ルール

### 500文字ごとに視覚的な休憩を入れる

**視覚要素の種類:**
1. まとめ枠（summary-box）
2. 画像・写真
3. YouTube埋め込み
4. 引用ボックス（blockquote）

---

## 📦 まとめ枠（summary-box）の使い方

### いつ使う？
- MVシーンの意味をまとめる時
- 複数の要素を箇条書きでまとめる時
- 「つまり」「要するに」というまとめの時

### 例：Bad Romanceの場合

**Before（文字だけ）:**
```
棺桶から出てくるシーンは、アーティストが「商品」として生み出される過程を表している。
ロシアンマフィアに売られるシーンは、レコード会社との契約、商品化を意味する。
最後に男を燃やすシーンは、システムへの反逆、復讐を表現している。
```

**After（まとめ枠）:**
```html
<div class="summary-box">
<strong>MVの象徴的なシーン</strong>

- **棺桶から出てくる** → アーティストが「商品」として生み出される過程
- **ロシアンマフィアに売られる** → レコード会社との契約、商品化
- **最後に男を燃やす** → システムへの反逆、復讐
</div>
```

## 🎨 まとめ枠のデザイン

### CSS実装
```css
.summary-box {
  background-color: #f1f5f9; /* 薄い青グレー */
  padding: 20px 24px;
  border-radius: 8px;
  border-left: 4px solid #64748b; /* 左に太いアクセント線 */
  margin: 24px 0;
  font-size: 1rem;
  line-height: 1.7;
}

.summary-box strong {
  display: block;
  margin-bottom: 12px;
  color: #1e293b;
  font-size: 1.0625rem;
}

.summary-box ul {
  margin: 0;
  padding-left: 1.5rem;
}

.summary-box li {
  margin-bottom: 8px;
}
```

## 📏 文字数カウントのガイドライン

### 1セクション（H2〜次のH2まで）

| 文字数 | 推奨アクション |
|--------|--------------|
| 0-300文字 | 何もしない（短いセクション） |
| 300-500文字 | 引用ボックス or まとめ枠を1つ |
| 500-800文字 | まとめ枠 + 画像、または2つの視覚要素 |
| 800文字以上 | H3で分割 + 各H3に視覚要素 |

---

## 📸 画像挿入のルール

### いつ使う？
- アーティストの写真
- アルバムカバー
- ライブ写真
- MV のスクリーンショット

### サイズルール
- **アルバムカバー**: 300px × 300px（正方形）
- **ライブ写真**: 600px × 400px（横長）
- **MV・映像**: YouTube埋め込み（560×315）

## 🎯 実装例（Bad Romance）

### 修正箇所1: MVのシーン解説（800文字）
**Before:** 長い段落が続く
**After:** まとめ枠を追加

```html
<div class="summary-box">
<strong>MVの象徴的なシーン</strong>

- **棺桶から出てくる** → アーティストが「商品」として生み出される過程
- **ロシアンマフィアに売られる** → レコード会社との契約、商品化
- **最後に男を燃やす** → システムへの反逆、復讐
</div>
```

### 修正箇所2: 歌詞の意味（600文字）
**Before:** 文字だけの説明
**After:** 引用ボックス + まとめ枠

```markdown
> I want your ugly, I want your disease

<div class="summary-box">
<strong>この歌詞の意味</strong>

完璧な愛ではなく、相手の欠点や暗い部分まで、すべてを受け入れたい——そんな強烈な渇望。
</div>
```

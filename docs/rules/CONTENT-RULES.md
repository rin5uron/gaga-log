# コンテンツ作成ルール

**作成日**: 2026-01-01  
**最終更新**: 2026-01-01

---

## リンクのルール

### 基本方針

**そのページの該当アーティストと該当曲はリンクなし**

- アーティストページでは、そのアーティストの曲へのリンクは付けない
- 記事ページでは、その記事の曲名へのリンクは付けない
- コラボ曲の場合、参加している全アーティストに個別にリンクを付ける

### アーティストページの「私のお気に入り曲・作品」セクション

**ルール:**
- そのアーティストの曲はリンクなし（テキストのみ）
- 他のアーティストの曲はリンクあり
- コラボ曲の場合、参加している全アーティストに個別にリンクを付ける

**例:**
```markdown
## 私のお気に入り曲・作品

このブログで紹介している曲：

- **Bad Romance** - 恋に壊れる瞬間を歌った、強烈な曲。（リンクなし）
- **Rain on Me** - [Ariana Grande](/artists/ariana-grande)とのコラボ。痛みを受け入れながら前に進む、力強いメッセージ。
- **[Telephone](/posts/telephone)** - [Beyoncé](/artists/beyonce)とのコラボ。自由を選ぶ勇気を与えてくれる曲。
```

### 記事ページの「関連記事」セクション

**ルール:**
- その記事の曲名はリンクなし（テキストのみ）
- 他の曲へのリンクは付ける
- コラボ曲の場合、参加している全アーティストに個別にリンクを付ける

**例:**
```markdown
### 関連記事

- **Rain on Me** - アリアナ・グランデとのコラボ。（リンクなし、この記事の曲名）
- [Telephone](/posts/telephone) - ビヨンセとのコラボ。
- [Lady Gaga](/artists/lady-gaga) - Lady Gagaの他の楽曲を探す
- [Ariana Grande](/artists/ariana-grande) - Ariana Grandeの他の楽曲を探す
```

### 本文中のリンク

**ルール:**
- 本文中で他の曲やアーティストに言及する場合は、リンクを付ける
- ただし、過度にリンクを付けすぎない（読みやすさを優先）

## アーティスト名の表記ルール

### 基本方針

**フロントマター・見出し・リンク: 英語表記（正式名称）**
**本文中: カタカナ表記（親しみやすく）**

### 表記の使い分け

#### 1. フロントマター（`artist`フィールド）

- **英語表記（正式名称）**を使用
- 例: `artist: "Lady Gaga"`, `artist: "Lady Gaga feat. Ariana Grande"`

#### 2. 見出し・セクションタイトル

- **英語表記（正式名称）**を使用
- 例: `## About <span class="section-subtitle">この曲について</span>`
- 本文中でアーティスト名を初めて言及する場合は、英語表記とカタカナ表記を併記

#### 3. 本文中

- **カタカナ表記**を使用（親しみやすく）
- 例: 「ガガ」「アリアナ・グランデ」「エド・シーラン」
- 初出時のみ、英語表記とカタカナ表記を併記
  - 例: 「レディー・ガガ（Lady Gaga）は...」

#### 4. リンクテキスト

- **英語表記（正式名称）**を使用
- 例: `[Lady Gaga](/artists/lady-gaga)`

#### 5. アーティストページの「私のお気に入り曲・作品」セクション

- **カタカナ表記**を使用（親しみやすく）
- 例: 「ガガとアリアナ・グランデとのコラボ」

### 表記例

| 場所 | 表記 | 例 |
|------|------|-----|
| フロントマター | 英語（正式名称） | `artist: "Lady Gaga"` |
| 見出し | 英語（正式名称） | `## About` |
| 本文中（初出） | カタカナ + 英語 | 「レディー・ガガ（Lady Gaga）は...」 |
| 本文中（2回目以降） | カタカナ | 「ガガは...」 |
| リンクテキスト | 英語（正式名称） | `[Lady Gaga](/artists/lady-gaga)` |
| アーティストページの説明 | カタカナ | 「ガガとアリアナ・グランデとのコラボ」 |

## コラボ曲の表記ルール

### フロントマター

- **区切り文字**: `&`, `feat.`, `with`, `/` を使用
- 例: `artist: "Lady Gaga feat. Ariana Grande"`

### 本文中

- **カタカナ表記**を使用
- 例: 「ガガとアリアナ・グランデとのコラボ」

### リンク

- **参加している全アーティストに個別にリンクを付ける**
- 例: 「[Lady Gaga](/artists/lady-gaga)と[Ariana Grande](/artists/ariana-grande)とのコラボ」

## セクションの文量について

### 現状の分析

現在のセクションの文量は適切です。

**各セクションの目安:**
- **About**: 2〜3段落（客観的情報 + 主観的感想）
- **Lyrics**: 3〜5段落（歌詞のポイントを複数説明）
- **Favorite Lines**: 2〜3段落（覚えたいフレーズを紹介）
- **Music Video**: 2〜3段落（MVの見どころ）
- **主観的な感想セクション**: 2〜4段落（個人的な感想）

### 文量のチェック方法

1. **読みやすさ**: 1段落が長すぎないか（3〜5文程度）
2. **情報量**: 各セクションに十分な情報があるか
3. **バランス**: 各セクションの文量が均等か

### 改善のポイント

- 1段落が長すぎる場合は、2段落に分割
- 情報が少ないセクションは、もう少し詳しく書く
- 情報が多すぎるセクションは、要点を絞る

## その他のルール

### 日付の表記

- フロントマター: `date: "YYYY-MM-DD"`（ISO形式）
- 本文中: 「YYYY年MM月DD日」（日本語形式）

### アルバム名・曲名の表記

- **英語表記**を使用（正式名称）
- 例: 「『Born This Way』」「『The Fame』」

### 引用の表記

- 歌詞の引用は `>` を使用
- 例:
```markdown
> I was born this way
```

### 参考文献の表記

- 記事の最後、関連記事の上に配置
- 公式サイト、Wikipedia、ストリーミングサービスへのリンクを記載

- [ ] そのページの該当アーティストと該当曲はリンクなし
- [ ] コラボ曲の場合、参加している全アーティストに個別にリンクを付けた
- [ ] フロントマター・見出し・リンクは英語表記（正式名称）
- [ ] 本文中はカタカナ表記（親しみやすく）
- [ ] 本文中の初出時のみ、英語表記とカタカナ表記を併記
- [ ] 各セクションの文量が適切か確認
- [ ] 参考文献を記載した

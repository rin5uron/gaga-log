## 1. テンプレート整合性チェック

### フロントマター

- [ ] `title`が設定されている
- [ ] `artist`が設定されている（楽曲の場合は必須）
- [ ] `date`が設定されている（YYYY-MM-DD形式）
- [ ] `type`が設定されている（"song" or "movie"）
- [ ] `relatedPosts`が設定されている（関連記事がある場合）
- [ ] 楽曲の場合：`song`, `album`, `year`が設定されている

### 画像・メディア

- [ ] YouTube埋め込み または 視聴リンク付き画像がある
- [ ] 画像のalt属性が設定されている
- [ ] ストリーミングリンクがある（楽曲の場合）

### 見出し構造

#### About セクション
- [ ] `## About <span class="section-subtitle">〇〇</span>` がある
- [ ] h3サブセクションがある（最低1つ）
- [ ] 各h3に `<span class="section-subtitle">〇〇</span>` がついている

#### メインコンテンツセクション
- [ ] **楽曲**: `## Lyrics`, `## Favorite Lines`, `## Music Video` など
- [ ] **映像作品**: `## Highlights` がある
- [ ] 各h2に `<span class="section-subtitle">〇〇</span>` がついている
- [ ] h3サブセクションがある
- [ ] 各h3に `<span class="section-subtitle">〇〇</span>` がついている

#### 主観的セクション
- [ ] **楽曲**: 主観的なセクションがある（例: `## Thoughts`）
- [ ] **映像作品**: `## After Watching`, `## Resonance` がある
- [ ] 各h2に魅力的なサブタイトルがついている
- [ ] h3サブセクションがある（推奨）
- [ ] 各h3に `<span class="section-subtitle">〇〇</span>` がついている

---

## 2. SEO要素チェック

### メタデータ（自動生成）
- [ ] タイトルが適切に生成されている
- [ ] descriptionが適切に生成されている
- [ ] キーワードが適切に設定されている

### コンテンツSEO

#### タイトル・見出し
- [ ] h1（記事タイトル）にターゲットキーワードが含まれている
- [ ] h2見出しが適切に構造化されている
- [ ] サブタイトルが魅力的でクリックを誘う内容になっている

#### 本文
- [ ] 冒頭100文字以内にキーワードが含まれている
- [ ] 適度にキーワードが自然に含まれている
- [ ] 段落が適切な長さ（3-5文程度）で読みやすい
- [ ] 視覚的な区切り（`---`）が適切に使われている

#### リンク
- [ ] 内部リンク（関連記事）が適切に設定されている
- [ ] アーティストページへのリンクが自動生成される
- [ ] 外部リンクがある場合、適切に設定されている
- [ ] 見出し内にはリンクが含まれていない

---

## 3. ユーザー体験チェック

### 読みやすさ
- [ ] 目次が自動生成される構造になっている
- [ ] h2とh3の階層が明確
- [ ] セクション間に適切な余白がある
- [ ] 文量が適切（各セクション2-5段落程度）

### コンテンツの質
- [ ] 個人的な視点・感想が含まれている
- [ ] 具体的な例やエピソードがある
- [ ] 読者に価値を提供する内容になっている
- [ ] 最後まで読みたくなる構成になっている

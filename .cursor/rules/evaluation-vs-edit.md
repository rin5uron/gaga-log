# 評価を聞かれたときは編集しない

## 認識のずれを防ぐ

- **「どう思う？」「どう？」「評価して」「どうかな？」** など、意見・評価を聞かれているときは、**意見・評価だけを返す。ファイルの作成・編集はしない。**
- ファイルを変えてよいのは、**「書いておいて」「変更して」「やって」「〇〇に反映して」** など、変更を依頼する言葉をユーザーが使ったときだけ。
- 提案や案を出したあと、「でどう？」と聞かれたら、それは「あなたの意見を聞いている」であり、「その案でメモを書き換えて」ではない。**「どう思う？」＝ 意見を言う。「〇〇して」＝ はじめて実行する。**

## 確認してから動く

- ユーザーが案を述べたあと、こちらの解釈が「変更の依頼」か「意見を求めているだけ」か迷ったら、**「意見だけ言いますか？ それともこの内容でメモを書き換えますか？」** と短く確認してから動く。

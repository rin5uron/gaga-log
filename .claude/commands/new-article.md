# 新規記事作成

`.claude/article-workflow.md` に記載された手順に従って、新しい記事を作成してください。

## 実行手順

1. ユーザーに曲名とアーティスト名を確認する
2. `.claude/article-workflow.md` の「2. 質問リスト」にある6つの質問をすべて聞く
3. Web検索で曲の基本情報（リリース年、アルバム名、制作背景）を調べる
4. `content/posts/[曲名].md` ファイルを作成
5. `article-workflow.md` のフォーマットルールに従って記事を生成
6. 作成完了後、ユーザーに確認を促す

## 重要

- `content/posts/template.md` を参考にフォーマットを統一
- フロントマターの `title` はプレースホルダーのまま残す
- タイトル（##）の前には必ず2行の空行を入れる
- 絵文字は使わない（明示的に依頼された場合を除く）
